# 职业健康分析技能

## When to Use
- 需要评估工作相关健康风险、人机工程问题或职业健康趋势时使用。
- 任务涉及久坐、视屏终端、倒班、重复性劳损或工作压力等职业风险分析。
- 用户请求职业健康评估、工作环境改进建议或职业病风险预警时使用。

## 核心功能

职业健康分析技能提供全面的职业健康数据分析功能，帮助用户追踪工作相关健康问题、识别职业健康风险、评估工作环境人机工程水平和优化职业健康。

**主要功能模块：**

1. **职业健康风险评估** - 久坐、视屏终端、倒班工作、重复性劳损、工作压力等多维度风险评估
2. **工作相关问题追踪** - 颈肩腰腿痛、眼疲劳、手腕不适等症状记录，不推断诊断
3. **人机工程评估** - 工作站、椅子、显示器、键盘、环境等全方位评估
4. **专业评估准备** - 根据暴露、症状和当地要求整理供职业健康专业人员评估的信息
5. **趋势分析** - 症状发展、改善效果、风险变化趋势
6. **关联分析** - 与睡眠、运动、心理健康、慢性病模块的关联分析
7. **个性化建议** - 工作姿势、休息提醒、设备建议、环境优化
8. **预警系统** - 高风险模式、症状恶化、职业病风险预警

## 触发条件

技能在以下情况下自动触发：

1. 用户使用 `/work trend` 查看职业健康趋势
2. 用户使用 `/work status` 查看综合健康状态
3. 用户使用 `/work recommend` 获取改进建议
4. 用户使用 `/work assess` 进行综合评估
5. 用户使用 `/work issue` 记录问题后的分析
6. 用户使用 `/work ergonomic` 进行人机工程评估后的分析

## 医学安全边界

**本技能不能做的事：**
- ❌ 不进行职业病诊断
- ❌ 不出具职业病诊断证明
- ❌ 不替代工作场所健康监护
- ❌ 不预测疾病发展
- ❌ 不处理急性健康危机

**本技能能做的事：**
- ✅ 描述工作暴露、症状和变化趋势
- ✅ 工作相关症状识别和追踪
- ✅ 人机工程评估和改进建议
- ✅ 识别需要向合格职业健康专业人员咨询的情况
- ✅ 工作环境改善建议
- ✅ 健康记录保存（就医时参考）
- ✅ 与其他健康数据的关联分析

## 执行步骤

### 第1步：数据读取

读取职业健康数据文件：
- `data-example/occupational-health-tracker.json` - 主职业健康档案

**数据验证：**
- 检查文件是否存在
- 验证数据结构完整性
- 确认有足够的数据点进行分析

### 第2步：职业健康观察

不得使用未经验证的固定权重、分数阈值或“低/中/高风险”等级来解释健康风险。记录原始观察，并保留数据来源、时间范围和缺失项。

#### 久坐与活动模式

- 每天久坐时间及其测量方式
- 休息和姿势变化的实际频率
- 每周活动记录
- 症状的部位、严重程度、持续时间和变化

#### 视屏终端观察

- 屏幕使用时长、连续使用时段和任务类型
- 休息、焦距变化和眨眼等习惯
- 照明、眩光、显示器位置和可调节性
- 眼部或头痛症状的时间、持续程度和变化

#### 综合整理

按久坐、视屏、倒班、重复性动作、噪声、粉尘/化学品和工作压力分别列出观察、趋势和不确定性。不得把不同暴露合并成一个未经验证的健康分数。

### 第3步：人机工程评估

#### 评估维度

描述椅子和工作台的可调节性、显示器位置、键盘鼠标布局、照明、噪声、温度，以及任务和个人需求。对每项记录“已观察”“未观察”或“信息不足”，并给出可验证的改进项。只有在采用明确命名、适用于该工作场景且由合格人员执行的现行人机工程方法时，才可报告该方法定义的分数。

### 第4步：专业评估与当地要求

不得按工作类型自行规定普遍适用的检查项目或固定频率。先确认国家/地区、岗位、实际暴露、雇主职业健康制度及当前主管机构指南。对于噪声、粉尘、化学品、倒班、重复性动作等暴露，整理暴露记录和症状，建议由雇主指定的职业健康服务、合格职业医师或当地主管机构确定是否需要健康监护以及具体周期。

### 第5步：关联分析

#### 睡眠-职业健康关联
- 倒班工作与睡眠质量的相关性
- 睡眠不足与工作相关症状的关系

#### 运动-职业健康关联
- 久坐工作与运动量的关系
- 运动与肌肉骨骼症状的关系

#### 心理健康-职业健康关联
- 工作压力与心理状态的关系
- 职业健康问题与心理症状的关联

### 第6步：生成报告

输出包括：
- 职业健康状况摘要
- 暴露与症状观察及趋势
- 工作相关问题分析
- 人机工程评估结果
- 需要专业评估或核对当地要求的事项
- 与其他健康因素的关联分析
- 预警信息（如适用）
- 个性化建议和行动计划

## 错误处理

### 数据文件不存在
```
错误：未找到职业健康数据文件
建议：请先使用 /work assess 命令创建数据
```

### 数据不足
```
警告：数据不足以进行趋势分析
建议：至少需要3次评估记录
```

### 需要及时专业评估的情况
```
⚠️ 观察到需要专业评估的症状或暴露

观察到：
- [列出症状、变化或暴露，并注明数据来源]

建议行动：
1. 如有急性或严重症状，联系当地急救或紧急医疗服务
2. 咨询合格的职业健康专业人员，由其评估诊断和工作相关性
3. 按当地制度与雇主职业健康服务核对暴露控制和工作调整流程
```
