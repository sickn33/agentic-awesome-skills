# 中医体质问卷整理

## When to Use

- 用户希望整理自己已填写的中医体质问卷或比较不同时间点的答案。
- 用户明确需要文化性/传统框架下的描述性摘要，而非医学诊断或治疗。
- 用户希望准备与合格医生或中医师讨论的问题。

## Safety Boundary（安全边界）

- 体质分类不是现代医学诊断，不用于排除疾病、判断预后或替代检查。
- 不提供方剂、单味药、克数、服法、疗程、加减、针灸、艾灸、拔罐或其他操作步骤。
- 不根据问卷推荐自行用药、停药、改变治疗或延误就医。
- 不声称食物、穴位或传统疗法能治疗具体疾病。
- 对孕期、儿童、慢性病、肝肾功能异常、过敏、手术前后或正在用药者，不给出个体化传统治疗建议。

出现急性或快速恶化症状时，停止问卷解释并建议使用当地正规医疗服务。传统分类不得覆盖红旗或专业诊断。

## 工作流程

1. **确认目的和同意**：说明输出仅为问卷整理，确认用户是否愿意使用这一传统框架。
2. **记录问卷版本**：保存来源、语言、日期、题目和用户原始答案；版本不明时不得声称使用标准量表。
3. **按声明规则计算**：只有在问卷附带完整、可追溯的计分说明时才复算；否则原样报告提供者给出的结果。
4. **描述而不诊断**：使用“该问卷将这些回答归为……”而非“你属于……”或“你患有……”。
5. **标注不确定性**：说明自报偏差、版本差异、分类重叠以及缺乏现代医学诊断效力。
6. **生成讨论清单**：列出用户关注的症状、正在使用的药物/草药、过敏和希望专业人员核实的问题，不给出用法用量。

## 输出格式

```markdown
## 问卷范围
- 来源/版本/日期：
- 缺失或不可核实信息：

## 描述性结果
- 问卷规则产生的分类：
- 支持该分类的原始回答：
- 重叠与不确定性：

## 安全与专业复核
- 不能由问卷推出：
- 需要向医生或合格中医师确认的问题：
```

## Limitations

- 本技能不验证中医体质理论的医学有效性，也不提供诊断或治疗。
- 草药和操作性疗法可能有副作用、污染、禁忌和药物相互作用，必须由合格专业人员结合个人情况评估。
- 不使用问卷结果替代循证筛查、实验室检查或紧急医疗判断。
