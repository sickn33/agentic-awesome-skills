# 心理健康趋势整理

## When to Use

- 用户希望整理自己的 PHQ-9、GAD-7 等已完成量表或情绪日记。
- 用户需要为医生、心理师或其他合格专业人员准备时间线和问题清单。
- 用户要比较不同时间点的数据，但不要求诊断、治疗或个人危机预测。

## Safety Boundary（安全边界）

- 不诊断心理疾病，不判定严重程度，不预测自杀、自伤或暴力风险。
- 不把量表总分、单项分数或自制权重转换成个人危机等级。
- 不建议开始、停止或调整药物、治疗、住院或安全计划。
- 不提供未经实时核验的热线号码；资源必须按用户所在地核验。
- 只处理完成任务所需的最少数据。先取得同意，不要求真实姓名或可识别信息。

## 紧急分流优先

如果用户表示正在实施或准备实施自伤/自杀、无法保证当下安全、出现严重意识混乱或对他人有紧迫危险：

1. 停止评分、趋势分析和长篇追问。
2. 用直接、支持性的语言建议立即联系当地急救服务、危机服务或可信赖且能在场的人。
3. 询问用户所在国家或地区，仅用于查找当前官方资源；无法核实时明确说明，不猜测号码。
4. 鼓励远离可用于伤害的物品并前往有其他人在场的安全地点，但不要承诺监控或替代急救人员。

当量表中出现与自伤有关的回答时，不以总分抵消该信号；直接提示需要由合格专业人员及时复核。

## 工作流程

1. **确认范围与同意**：说明这是非诊断性整理，确认时间段、数据来源和期望输出。
2. **保留原始含义**：记录日期、量表名称、用户填写的分数或原话；不要补造缺失值。
3. **检查可比性**：确认量表版本、语言、填写方式和时间间隔是否一致。不可比时分开呈现。
4. **描述变化**：报告绝对变化、方向和缺失数据，不给出因果结论。将睡眠、事件或治疗变化标为同期因素而非原因。
5. **标注不确定性**：区分用户事实、主观感受、计算结果和推测。
6. **准备专业复核**：生成简短时间线、需要核对的异常值、用户最关心的问题和需要专业人员解释的项目。

## 输出格式

```markdown
## 范围与数据质量

- 数据来源和版本：
- 缺失或不可比之处：

## 观察到的变化

|---|---|---|---|

## 不确定性
- 已知：
- 尚未核实：
- 不能由这些数据推出：

## 专业复核问题
1. ...
```

## Limitations

- 量表是筛查或追踪工具，不等于诊断，也不能单独决定治疗或危机处置。
- 相关性不证明因果；药物、治疗和生活事件只能作为待专业核实的背景。
- 紧急情况优先使用用户所在地的实时官方急救与危机资源。
