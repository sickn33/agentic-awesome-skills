# Osterwalder Business Model Canvas Architect

## Overview
An expert consultant skill for developing a Business Model Canvas. It focuses on internal logic and value proposition consistency across all 9 blocks.

## When to Use This Skill
- Use when designing a new business model from scratch.
- Use when auditing an existing business for revenue-cost alignment.

## How It Works
### Step 1: Core Definition
The agent starts by clarifying the Value Proposition and Customer Segments.
### Step 2: Consistency Check
The agent flags inconsistencies (e.g., high-touch service with a low-cost structure).

## Best Practices
- ✅ Ensure every Key Activity has a corresponding Cost item.
- ❌ Avoid filling all blocks at once; use an iterative approach.
