# 康复数据趋势整理

## When to Use

- 用户已有医生、物理治疗师或其他合格临床人员制定的康复计划。
- 用户希望整理疼痛、活动度、功能测试、训练完成情况或症状日志。
- 用户需要为复诊准备时间线，而不是获得新的训练处方或阶段判断。

## Safety Boundary（安全边界）

- 不诊断损伤、并发症或康复阶段。
- 不根据完成率、单一疼痛分数或自制准备度分数建议升级阶段、增加负荷或继续训练。
- 不开始、停止或调整药物、器械、训练动作、频率、阻力、活动范围或负重状态。
- 只对照临床人员明确提供的目标和停止条件；缺少标准时标为待复核。

出现以下情况时停止常规分析并建议及时或紧急联系当地医疗服务：新发或加重的胸痛/呼吸困难、意识异常、无法控制的出血、伤口感染迹象、突然失去力量或感觉、大小便控制改变、剧烈或快速恶化的疼痛、术后医生标明的红旗。不要用总分抵消红旗。

## 工作流程

1. **确认计划来源**：记录临床人员、计划版本、手术/损伤背景、允许和禁止事项、复诊日期。
2. **确认同意与最少数据**：只收集完成摘要所需的数据，去除不必要的身份信息。
3. **标准化时间线**：按日期记录用户提供的疼痛、功能、活动度、训练和症状；保留测量单位和测试方法。
4. **检查可比性**：不同设备、动作、评估者或条件下的数据不得直接合并。
5. **描述趋势**：报告方向、波动、缺失和异常点；不推断组织愈合或训练准备度。
6. **对照临床标准**：仅标示“达到/未达到/未知”，不自行改变阈值或给出进阶建议。
7. **生成复诊摘要**：列出红旗、计划偏离、用户问题以及需要临床人员决定的阶段/强度事项。

## 输出格式

```markdown
## 计划与范围
- 临床计划/版本：
- 明确目标和停止条件：

## 趋势
| 日期 | 用户提供的测量 | 条件 | 变化 | 可比性 |
|---|---|---|---|---|

## 需要临床复核
- 红旗或异常变化：
- 阶段、强度、药物或器械问题：
```

## Limitations

- 自报疼痛和家庭测量不能替代体格检查或术后评估。
- 相关变化不证明康复措施导致结果。
- 所有训练阶段、负荷和治疗调整由负责的临床人员决定。
