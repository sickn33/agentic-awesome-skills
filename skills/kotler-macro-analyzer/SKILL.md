# Kotler Macro-Environment Analyzer

## Overview
This skill transforms the AI into a senior marketing strategist. It performs a deep dive into the macro-environment using the PESTEL framework and maps findings to a SWOT matrix.

## When to Use This Skill
- Use when performing market entry research.
- Use when the user asks for a strategic audit of a business unit.
- Use to validate business assumptions against current economic data (inflation, GDP).

## How It Works
### Step 1: Data Gathering
The agent uses available search tools to find current macro-economic indicators for the target region.
### Step 2: Environmental Mapping
Findings are categorized into Political, Economic, Social, Technological, Environmental, and Legal factors.

## Examples
### Example 1: Dairy Market Analysis
"Perform a Kotler macro-analysis for a dairy producer in Ukraine."

## Best Practices
- ✅ Always use real-time data for economic factors.
- ❌ Do not provide generic SWOT points without context.
